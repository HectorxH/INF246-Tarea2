SOBRE MAKEFILE
(los comandos se anotan en comillas para diferenciarlas del texto simple)
	☆ Ingresar "make" en la consola para ejecutar el makefile.
	☆ Ingresar "./main.out" para ejecutar el código.
	☆ Ingresar "make clean" en la consola para borrar los archivos .o y .out
+---------------------------------------------------------+
INSTRUCCIONES DE USO
	☆ El juego parte de la selección de número de turno, de 1 a 4 para el
	jugador principal. Los turnos de los demás jugadores se asignan en base
	a esta elección de manera automática.
	☆ Para continuar al siguiente turno, se debe ingresar por la consola 
	cualquier número (por ejemplo, 1).
	☆ Para activar el efecto especial (? o ??) se debe ingresar 1 ante la
	petición por consola. En caso contrario, ingresar 0.
	☆ La flecha <-- o --> indica el orden de turnos.
	☆ El orden de tablero es siempre de sta a fin.
	☆ Al invertir el orden de tablero, se intercambian las posiciones de
	sta y fin.
	☆ No se imprimen las flechas en el tablero pues se asume la dirección
	como implícita en base a lo mencionado antes.	
+---------------------------------------------------------+
INDICACIONES:
	☆ Los demás jugadores siempre eligen activar el efecto ? o ??.
	☆ Si el jugador principal no es el jugador actual, en caso de que algún
	otro jugador active el efecto de RETROCEDER UN PASO o RETROCEDER DOS PASOS 
	y el jugador principal
	caiga en la celda ? o ??, no se activa ningún efecto.
	☆ En caso de empate, se elige el jugador de menor id.
